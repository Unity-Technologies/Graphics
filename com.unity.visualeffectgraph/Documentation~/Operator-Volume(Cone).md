# Volume (Cone)

Menu Path : **Operator > Math > Geometry > Volume (Cone)**

The **Volume (Cone)** Operator calculates the volume of a cone. This is equal to **1/3 Ï€ h(R<sup>2</sup> + R r + r<sup>2</sup>)** where **h** is the height of the cone, **R** is the radius of the base, and **r** is the radius of the cap.

##  properties

| **Input** | **Type** | **Description**                                  |
| --------- | -------- | ------------------------------------------------ |
| **Cone**  | Cone     | The cone this Operator calculates the volume of. |

| **Output** | **Type** | **Description**         |
| ---------- | -------- | ----------------------- |
| **Volume** | float    | The volume of the cone. |
