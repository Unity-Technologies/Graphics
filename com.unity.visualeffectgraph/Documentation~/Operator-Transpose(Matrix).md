# Transpose (Matrix)

Menu Path : **Operator > Math > Geometry > Transpose (Matrix)**

The **Transpose (Matrix)** Operator flips a matrix across its diagonal, swapping the matrixâ€™s column values and its row values. This is the same operation as [Matrix4x4.transpose](https://docs.unity3d.com/ScriptReference/Matrix4x4-transpose.html).

## Operator properties

| **Input**  | **Type** | **Description**                      |
| ---------- | -------- | ------------------------------------ |
| **Matrix** | Matrix   | The Matrix this Operator transposes. |

| **Output** | **Type** | **Description**                      |
| ---------- | -------- | ------------------------------------ |
| **Matrix** | Matrix   | The result of the transposed Matrix. |
