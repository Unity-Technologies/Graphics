# Direction

Menu Path : **Operator > Inline > Direction**

The **Direction** inline Operator allows you to store an instance of a [Direction](Type-Direction.md). This is one of the Visual Effect Graph's advanced [Types](VisualEffectGraphTypeReference.md). To modify a value in this inline Operator, either set the value explicitly in the Operator body, or connect it to a compatible output.

[!include[](Snippets/Operator-InlineIntro.md)]


## Operator properties

| **Input** | **Type** | **Description**                                              |
| --------- | -------- | ------------------------------------------------------------ |
| **Input** | Direction   | The value of the Operator. You can either set this in the Operator's body, or connect to it from the output of another Operator. |

| **Output** | **Type** | **Description**            |
| ---------- | -------- | -------------------------- |
| **Output** | Direction   | The value of the Operator. |
