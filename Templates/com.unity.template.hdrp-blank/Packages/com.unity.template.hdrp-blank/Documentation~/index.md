# About HDRP Blank Template

*This template includes the settings and assets you need to start creating with the High Definition Render Pipeline (HDRP).*

# Using the HDRP Blank Template

The settings of this template are preconfigured for the High Definition Render Pipeline.

This template also contains default assets for creating with the HDRP.

This template does not contain presets or example content.

Packages included:

- [High Definition RP](https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@7.5/manual/index.html)


# Technical details

## Requirements

This version of the HDRP Blank Template is compatible with the following versions of the Unity Editor:

* 2019 LTS and above (recommended)
